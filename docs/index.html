Advanced Network Service Testing Environment

ANSTE is a tool to easily create, deploy and test network services on complex scenarios using virtual machines.

Install ANSTE on Ubuntu

sudo add-apt-repository ppa:jacalvo/anste
sudo apt-get update
sudo apt-get install anste

Using ANSTE

anste -help
